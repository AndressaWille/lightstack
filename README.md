# Lightstack

Tools for building and processing multi-filter astrophysical datacubes.

## Features

- Cropping FITS images  
- Building photometric datacubes  
- PSF matching  

(Currently supports JWST and HST)

## Installation

```bash
pip install lightstack
```bash
